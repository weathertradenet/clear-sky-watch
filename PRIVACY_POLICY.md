# Privacy policy draft

Effective date: 2026-05-24

Clear Sky Watch is a Chrome Extension demo owned by [Demetra](https://www.pappers.fr/entreprise/demetra-941049322).

## What the extension does

The extension reads destination and trip-date information visible on supported travel websites in order to let the user save destinations and compare weather forecasts or typical monthly weather statistics.

## Data stored locally

The extension may store the following information locally in the user's browser:

- Saved destination names
- Trip dates detected from travel pages
- Optional WeatherAPI.com or OpenWeather API keys entered by the user
- Basic comparison preferences or cached results, if implemented

This data is stored using Chrome local extension storage.

## Data sent to third-party services

To provide weather comparison, the extension may send destination names or coordinates to third-party geocoding and weather APIs, including Open-Meteo, OpenStreetMap Nominatim, Photon / Komoot, MET Norway, WeatherAPI.com, and OpenWeather.

The extension does not intentionally send booking prices, passenger names, payment information, account credentials, or reservation details.

## Data not collected by [Demetra](https://www.pappers.fr/entreprise/demetra-941049322) in this demo

In the current demo version, [Demetra](https://www.pappers.fr/entreprise/demetra-941049322) does not operate a backend server for collecting extension user data. The extension works locally in the browser and communicates directly with weather/geocoding APIs.

## API keys

If the user enters optional API keys, those keys are stored locally in the browser. For a public production version, a backend proxy or other secure key-management system is recommended.

## User control

Users can clear saved destinations inside the extension or remove the extension from Chrome.
