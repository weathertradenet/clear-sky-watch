# Release checklist

Before public release:

- Review licence with a qualified lawyer.
- Review privacy policy with a qualified lawyer.
- Confirm all API terms allow the planned usage.
- Remove any hardcoded private API keys.
- Create a public privacy policy URL.
- Create support contact email.
- Prepare Chrome Web Store screenshots.
- Prepare short and long descriptions.
- Test destination/date detection on top-priority travel websites.
- Test on Chrome stable.
- Confirm Manifest V3 compliance.
- Check that no remotely hosted executable code is used.
- Confirm extension permissions are as narrow as practical.
- Decide visibility: unlisted for demo, public for launch, private for internal use.
